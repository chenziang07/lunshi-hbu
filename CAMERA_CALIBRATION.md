## 方法一：简易焦距标定（推荐）

### 原理
使用已知尺寸的 AprilTag 和已知距离，通过相似三角形原理计算焦距：
```
焦距 = (图像中的像素宽度 × 实际距离) / AprilTag 实际尺寸
```

### 准备工作

1. **打印 AprilTag**
   - 下载：https://github.com/AprilRobotics/apriltag-imgs
   - 推荐使用 tag36h11 家族（如 tag36_11_00000.png）
   - 打印在 A4 纸上
   - **重要**：用尺子测量打印出的黑色方块边长（不含白边）

2. **准备尺子**
   - 用于测量 AprilTag 到相机镜头的距离
   - 建议测量 3-5 个不同距离（如 20cm, 30cm, 50cm）

### 标定步骤

1. **运行标定脚本**
   ```bash
   python calibrate_focal_length.py
   ```

2. **更新 AprilTag 尺寸**（如果需要）
   - 在 `config.py` 中设置 `APRILTAG_SIZE` 为实际测量的尺寸（单位：米）
   - 例如：如果测量是 5cm，设置 `APRILTAG_SIZE = 0.05`

3. **采集测量数据**
   - 将 AprilTag 放在某个距离（如 30cm）
   - 用尺子测量 AprilTag 到相机镜头的距离
   - 确保 AprilTag 正面朝向相机，完全在画面内
   - 按 **[空格]** 键
   - 输入实际距离（单位：厘米）

4. **多次测量（推荐）**
   - 在不同距离重复步骤 3（建议 3-5 次）
   - 距离范围：20cm - 100cm
   - 程序会自动计算平均值

5. **完成标定**
   - 按 **[q]** 键退出
   - 程序显示计算结果
   - 选择 `y` 自动更新 `config.py`

### 注意事项

- ✅ 确保 AprilTag 完全在画面内
- ✅ 测量距离时从相机镜头到 AprilTag 表面
- ✅ AprilTag 尺寸是指黑色方块外边长，不包括白边
- ✅ 多次测量可以提高精度
- ⚠️ 如果变异系数 > 5%，建议重新测量

## 方法二：完整相机标定

## 准备工作

### 1. 打印棋盘格标定板

下载并打印标准的棋盘格图案：
- 推荐尺寸：A4 纸
- 棋盘格配置：**9x6 内角点**（即 10x7 个方块）
- 方块边长：**2.5cm**

**下载地址：**
- OpenCV 官方：https://github.com/opencv/opencv/blob/master/doc/pattern.png
- 或者搜索 "opencv checkerboard pattern 9x6"

**打印要求：**
- 使用白纸打印，不要缩放
- 打印后测量方块边长，确保是 2.5cm
- 如果不是 2.5cm，修改 `calibrate_camera.py` 中的 `SQUARE_SIZE` 参数
- 将棋盘格贴在平整硬板上（避免弯曲）

### 2. 安装依赖

确保已安装 OpenCV：
```bash
pip install opencv-python numpy
```

## 标定步骤

### 1. 运行标定脚本

```bash
python calibrate_camera.py
```

### 2. 采集图像

程序会打开相机窗口，显示实时画面：

- **绿色提示 "Found!"**：表示检测到棋盘格，可以采集
- **红色提示 "Searching..."**：表示未检测到，调整棋盘格位置

**操作：**
- 按 **[空格]** 采集当前图像
- 按 **[q]** 退出

**采集技巧：**
- 需要采集至少 **15 张**不同角度的图片
- 从不同位置、角度、距离拍摄棋盘格
- 确保棋盘格**完全在画面内**
- 避免模糊、反光、遮挡
- 覆盖画面的不同区域（中心、边缘、角落）

**推荐采集角度：**
1. 正面中心（3-4 张，不同距离）
2. 左侧倾斜（2-3 张）
3. 右侧倾斜（2-3 张）
4. 上方倾斜（2-3 张）
5. 下方倾斜（2-3 张）
6. 四个角落各 1 张

### 3. 等待标定完成

采集足够图片后，程序会自动开始标定，输出类似：

```

============================================================
```

## 标定质量评估

**重投影误差**：
- **< 0.5 像素**：优秀
- **0.5 - 1.0 像素**：良好
- **> 1.0 像素**：需要重新标定

如果误差过大，可能原因：
- 棋盘格不平整（弯曲）
- 采集图片质量差（模糊、反光）
- 采集角度不够多样
- 方块尺寸测量不准确

## 常见问题

### Q1: 检测不到棋盘格（一直显示红色）

**解决方法：**
- 确保棋盘格完全在画面内
- 增加光照，避免阴影
- 调整距离（不要太近或太远）
- 确保棋盘格是 9x6 内角点配置
- 如果使用其他尺寸，修改 `CHECKERBOARD` 参数

### Q2: 方块尺寸不是 2.5cm

**解决方法：**
- 测量实际打印的方块边长（单位：米）
- 修改 `calibrate_camera.py` 中的 `SQUARE_SIZE` 参数
- 例如：如果是 3cm，改为 `SQUARE_SIZE = 0.03`

### Q3: 相机分辨率不同

**解决方法：**
- 修改 `calibrate_camera.py` 中的 `CAMERA_WIDTH` 和 `CAMERA_HEIGHT`
- 必须与实际使用时的分辨率一致

### Q4: 标定后距离测量还是不准

**可能原因：**
- AprilTag 尺寸设置错误（检查 `APRILTAG_SIZE`）
- AprilTag 尺寸是指**黑色方块外边长**，不包括白边
- 相机焦距变化（如果有自动对焦功能）
